# cosmic-themes-extension
The Sol, Terra, and Luna themes of real-time WebGL are extensions for ChatGPT Chrome web version.（实时WebGL的Sol、Terra和Luna主题，适用于ChatGPT-Chrome网页版的扩展程序。）
# Cosmic Themes

一款适用于 ChatGPT 网页端的实时 WebGL 宇宙视觉扩展。

它会在 ChatGPT 页面底层实时渲染太阳、地球和月球主题，并根据当前选择的模型自动切换视觉效果。

目前包含三套主题：

- **Sol**：太阳、日冕、表面能量、磁场轨道与金色粒子
- **Terra**：地球、大气辉光、蓝色轨道与深空粒子
- **Luna**：月球、冷色月晕、银灰粒子与月尘效果

> 本项目是独立开发的非官方视觉扩展，与 OpenAI 无隶属、合作或认可关系。

# 安装扩展

本扩展目前通过 GitHub 分发，需要使用 Chrome 开发者模式手动导入。

## 第一步：解压安装包

将下载的 ZIP 解压到一个固定目录，例如：

```text
D:\ChromeExtensions\CosmicThemes
```

解压后的正确目录结构应类似：

```text
CosmicThemes/
├─ manifest.json
├─ README.md
├─ content/
│  ├─ main.js
│  ├─ renderer.js
│  └─ theme.css
├─ popup/
│  ├─ popup.html
│  ├─ popup.css
│  └─ popup.js
└─ icons/
   ├─ icon16.png
   ├─ icon32.png
   ├─ icon48.png
   └─ icon128.png
```

必须确认打开该文件夹后，可以直接看到：

```text
manifest.json
```

如果需要再进入一层文件夹才能看到 `manifest.json`，安装时应选择里面那一层。

安装后不要随意删除或移动这个文件夹，否则 Chrome 可能无法继续读取扩展文件。

---

## 第二步：打开扩展管理页面

在 Chrome 地址栏输入：

```text
chrome://extensions/
```

按下回车。

---

## 第三步：开启开发者模式

在扩展管理页面右上角打开：

```text
开发者模式
```

打开后，页面上方会显示：

```text
加载已解压的扩展程序
打包扩展程序
更新
```

---

## 第四步：加载扩展

点击：

```text
加载已解压的扩展程序
```

选择刚才解压后的扩展文件夹。

必须选择直接包含：

```text
manifest.json
```

的目录。

加载成功后，扩展列表中会出现：

```text
Cosmic Themes
```

---

## 第五步：固定扩展图标

点击 Chrome 地址栏右侧的拼图图标。

在扩展列表中找到：

```text
Cosmic Themes
```

点击旁边的图钉按钮，将扩展固定到浏览器工具栏。

之后可以直接点击扩展图标打开设置面板。

---

## 第六步：打开并刷新 ChatGPT

打开：

```text
https://chatgpt.com/
```

Windows 用户按：

```text
Ctrl + Shift + R
```

macOS 用户按：

```text
Command + Shift + R
```

强制刷新页面后，扩展会把 WebGL 宇宙场景加载到 ChatGPT 页面中。

如果已经打开了多个 ChatGPT 标签页，建议全部刷新。

# 基本使用方式

点击浏览器工具栏中的 Cosmic Themes 图标，即可打开扩展设置面板。

扩展提供两种主题切换方式：

1. 跟随模型自动切换
2. 手动固定主题

---

# 自动跟随模型

打开设置面板中的：

```text
跟随模型
```

扩展会检测 ChatGPT 当前模型选择器中的可见名称。

主题对应关系：

| 模型名称 | 主题 |
|---|---|
| Sol | 太阳主题 |
| Terra | 地球主题 |
| Luna | 月球主题 |

扩展可以识别类似名称（仅限Chatgpt work模式）：

```text
Sol
5.6 Sol
GPT-5.6 Sol
gpt-5.6-sol

Terra
5.6 Terra
GPT-5.6 Terra
gpt-5.6-terra

Luna
5.6 Luna
GPT-5.6 Luna
gpt-5.6-luna
```

当模型名称发生变化时，扩展会自动进行连续镜头转场。

扩展不会通过聊天内容判断模型，也不会读取用户发送的问题和回答内容。

---

# 手动选择主题

如果当前 ChatGPT 页面没有显示 Sol、Terra 或 Luna，或者自动识别没有生效，可以手动选择主题。

操作方法：

1. 点击 Cosmic Themes 扩展图标
2. 关闭“跟随模型”
3. 选择 Sol、Terra 或 Luna
4. 页面会立即切换到对应主题

也可以保持“跟随模型”开启，同时选择一个备用主题。

当扩展无法识别当前模型名称时，会使用备用主题。

# 三套主题说明

## Sol｜太阳主题

Sol 主题包含：

- 动态太阳表面
- 金橙色高温纹理
- 日冕发光
- 星球表面粒子
- 磁场轨道
- 金色环绕粒子
- 深空星尘
- 缓慢自转效果

Sol 整体亮度较高，建议不要同时将亮度、发光和粒子强度全部调到最大。

推荐设置：

```text
场景亮度：85%—100%
星球大小：85%—105%
粒子强度：70%—100%
发光强度：85%—115%
环绕密度：65%—95%
粒子速度：80%—105%
```

---

## Terra｜地球主题

Terra 主题包含：

- 蓝色轨道粒子
- 环绕光线
- 深空星尘
- 缓慢自转效果

推荐设置：

```text
场景亮度：90%—110%
星球大小：90%—110%
粒子强度：85%—120%
发光强度：95%—125%

粒子速度：85%—110%
```

---

Luna 主题包含：

- 缓慢轨道效果
- 深空背景
- 安静缓慢的自转效果

推荐设置：

```text
场景亮度：85%—105%
星球大小：90%—110%
粒子强度：70%—100%
发光强度：80%—110%
环绕密度：60%—90%
粒子速度：65%—95%
```

# 设置功能说明

## 启用扩展

控制 Cosmic Themes 是否在 ChatGPT 页面生效。

开启后：

- WebGL 星球显示
- 页面主题颜色生效
- 动态粒子开始运行
- 自动模型检测生效

关闭后：

- WebGL 背景隐藏
- 动画停止
- 页面恢复接近原始 ChatGPT 外观

关闭扩展后，建议刷新一次 ChatGPT 页面。

---

## 跟随模型

开启后，扩展会根据当前模型名称自动切换主题。

```text
Sol → 太阳
Terra → 地球
Luna → 月球
```

关闭后，扩展固定使用手动选择的主题。

---

## 连续动效

控制页面中的动态效果，包括：

- 星球自转
- 星球表面动画
- 粒子运动
- 环绕轨道运动
- 日冕、大气和月晕动画
- 鼠标轻微视差
- 模型切换镜头动画

关闭后：

- 星球保持静态
- 粒子停止移动
- 主题仍然可以切换
- 设备资源占用会降低

---

## 场景亮度

控制整个 WebGL 场景的总体亮度。

它会影响：

- 星球主体亮度
- 星球表面细节
- 环绕光线
- 粒子亮度
- 星球边缘辉光

数值过高可能影响文字阅读。

建议范围：

```text
85%—110%
```

---

## 星球大小

控制星球在页面中的显示尺寸。

数值越高：

- 星球越大
- 表面细节更明显
- 占据页面空间更多

数值越低：

- 星球更远
- 页面更安静
- 更适合长时间阅读

建议范围：

```text
80%—115%
```

---

## 粒子强度

控制粒子整体视觉强度，包括：

- 星球表面粒子
- 星球外围粒子
- 环绕粒子
- 背景星尘
- 轨道粒子

粒子强度过高可能造成画面过亮或视觉干扰。

Sol 主题建议保持在中等范围。

---

## 发光强度

控制以下效果的发光程度：

- Sol 日冕
- Terra 大气辉光
- Luna 冷色月晕
- 粒子外发光
- 星球边缘光
- 轨道粒子光晕

数值越高，星球越明亮。

性能较低的电脑建议适当降低。

---

## 环绕密度

控制星球周围轨道和环绕粒子的数量。

数值越高：

- 轨道数量更明显
- 环绕粒子更密集
- 视觉效果更丰富
- GPU 使用率可能提高

建议根据电脑性能调节。

---

## 粒子速度

控制动态粒子的运动速度，包括：

- 星球表面粒子
- 轨道粒子
- 环绕粒子
- 背景星尘
- 部分日冕和大气动画

推荐使用中低速度，过高会降低自然感。

---

## 水平位置

控制星球在页面中的左右位置。

```text
负值 → 星球向左移动
正值 → 星球向右移动
```

可以根据显示器尺寸、侧边栏宽度和个人使用习惯进行调整。

---

## 垂直位置

控制星球在页面中的上下位置。

```text
负值 → 星球向上移动
正值 → 星球向下移动
```

建议避免让星球主体挡住输入框和主要聊天内容。

---

## 转场时长

控制 Sol、Terra、Luna 之间的镜头切换时间。

数值较低：

- 切换更快
- 操作效率更高
- 可能显得稍微直接

数值较高：

- 转场更柔和
- 镜头运动更自然
- 等待时间更长

推荐范围：

```text
2800ms—4200ms
```

---

## 文字保护

控制聊天阅读区域的暗色保护层。

提高文字保护：

- 背景对正文影响更小
- 文字更加清晰
- 星球效果会稍微减弱

降低文字保护：

- 星球显示更加明显
- 背景视觉效果更强
- 可能影响文字阅读

推荐范围：

```text
65%—85%
```

---

## 整体视觉强度

控制扩展的整体视觉存在感。

它会综合影响：

- 场景透明度
- 背景氛围
- 星球存在感
- 粒子视觉强度
- 主题颜色明显程度

建议先使用默认值，再根据显示器亮度调整。

---

## 渲染质量

提供四种模式：

### 自动

扩展根据设备性能自动选择质量。

如果持续检测到帧率不足，会自动降低渲染质量。

普通用户建议使用这一模式。

### 高

适合：

- 独立显卡
- 性能较好的台式机
- 高性能笔记本

特点：

- 粒子更多
- 表面细节更丰富
- 渲染分辨率更高

### 平衡

适合普通办公电脑和大部分笔记本。

在视觉质量和性能之间取得平衡。

### 省电

适合：

- 集成显卡
- 低功耗笔记本
- 高分辨率显示器
- 浏览器同时打开多个页面

特点：

- 减少粒子数量
- 降低渲染分辨率
- 降低设备资源占用

# 常见问题

## 1. Chrome 提示“清单文件缺失或不可读取”

说明安装时选择了错误的文件夹。

请选择直接包含以下文件的目录：

```text
manifest.json
```

不要选择 ZIP 文件，也不要选择外层空目录。

---

## 2. 点击“加载已解压的扩展程序”后没有反应

请检查：

- ZIP 是否已经解压
- 是否选择了正确目录
- `manifest.json` 是否存在
- 文件是否被杀毒软件隔离
- Chrome 是否开启开发者模式

---

## 3. 安装后 ChatGPT 页面没有变化

依次检查：

1. 扩展是否已经启用
2. 扩展设置中的“启用扩展”是否打开
3. 当前网址是否为 `https://chatgpt.com/`
4. 是否强制刷新过页面
5. 是否打开了其他旧标签页
6. 浏览器是否支持 WebGL2

可以按：

```text
Ctrl + Shift + R
```

重新加载页面。

---

## 4. 主题没有跟随模型自动切换

可能原因：

- 当前模型名称不包含 Sol、Terra 或 Luna
- ChatGPT 页面结构发生更新
- 模型选择器尚未加载完成
- 网页没有强制刷新
- “跟随模型”开关没有开启

解决方法：

1. 打开扩展面板
2. 关闭“跟随模型”
3. 手动选择 Sol、Terra 或 Luna

---

## 5. 星球显示太大或挡住文字

打开扩展设置：

- 降低“星球大小”
- 调节“水平位置”
- 调节“垂直位置”
- 提高“文字保护”
- 降低“整体视觉强度”

---

## 6. Sol 主题太亮

建议降低：

- 场景亮度
- 粒子强度
- 发光强度
- 环绕密度
- 整体视觉强度

推荐先使用：

```text
场景亮度：90%
粒子强度：80%
发光强度：95%
环绕密度：75%
```

---

## 7. 页面卡顿或电脑风扇转速升高

尝试以下操作：

1. 将渲染质量改为“平衡”或“省电”
2. 降低粒子强度
3. 降低环绕密度
4. 降低发光强度
5. 关闭连续动效
6. 关闭其他动态网页扩展
7. 减少同时打开的 ChatGPT 标签页

---

## 8. 输入框、消息框或侧边栏颜色异常

通常是多个网页主题扩展发生冲突。

请关闭：

- 旧版 Cosmic Themes
- 其他 ChatGPT 主题扩展
- 强制网页深色模式扩展
- 修改网页颜色的浏览器扩展

确保扩展列表中只启用了一个 Cosmic Themes。

然后强制刷新 ChatGPT。

---

## 9. 切换模型时动画不流畅

可以尝试：

- 将渲染质量改为“平衡”
- 降低粒子强度
- 降低环绕密度
- 将转场时长提高到 3200ms 以上
- 关闭其他高占用网页
- 更新 Chrome 浏览器

---

## 10. Chrome 显示“由开发者模式加载的扩展程序”

这是因为本扩展通过 GitHub 分发，并通过开发者模式安装。

这是正常提示，不代表扩展安装失败。

扩展源码公开在 GitHub，用户可以自行检查扩展代码和权限。

# 浏览器兼容性

推荐使用：

- Google Chrome
- Microsoft Edge
- 其他基于 Chromium 的浏览器

本扩展需要浏览器支持：

```text
Manifest V3
WebGL2
chrome.storage.local
```

不保证兼容：

- Firefox
- Safari
- 较旧版本的 Chrome
- 禁用 WebGL 的浏览器环境

---

# 扩展权限说明

扩展仅申请：

```text
storage
https://chatgpt.com/*
```

## storage

用于将以下设置保存在浏览器本地：

- 当前主题
- 是否跟随模型
- 亮度
- 星球大小
- 星球位置
- 粒子强度
- 发光强度
- 动效设置
- 渲染质量

## chatgpt.com 访问权限

用于：

- 向 ChatGPT 页面注入 WebGL 场景
- 应用视觉主题
- 检测模型选择控件中的名称

不会用于读取聊天正文、输入内容或网络请求。

# 隐私说明

本扩展不会收集、记录或上传：

- 聊天正文
- 用户输入内容
- ChatGPT 回答内容
- 账号密码
- Cookie
- 浏览历史
- 个人文件
- 网络请求
- 鼠标轨迹
- 账号信息

所有视觉设置均通过：

```text
chrome.storage.local
```

保存在用户自己的浏览器中。

本扩展不包含：

- 远程脚本
- 广告代码
- 数据统计代码
- 第三方追踪代码
- 远程字体
- 用户行为分析
